V.A. Discretas

Función de masa de probabilidad

Dada una variable aleatoria discreta \[X\]:

\[ f(x_i) = p_i = P(X=x_i) \]

Ejemplo

Definimos la v.a. discreta \[X = \text{"Número de caras obtenidas al lanzar 3 monedas."}\]

  • \[ p_0 = f(0) = P(X=0) = \frac{1}{8} \]
  • \[ p_1 = f(1) = P(X=1) = \frac{3}{8} \]
  • \[ p_2 = f(2) = P(X=2) = \frac{3}{8} \]
  • \[ p_3 = f(3) = P(X=3) = \frac{1}{8} \]

Propiedades

  • \[f(x_i) \in [0, 1]\] (es una propiedad)
  • \[ \sum_i f(x_i) = 1 \]

V.A. Discretas

Función de distribución

Dada una variable aleatoria discreta \[X\]:

\[ F(x_i) = P(X \le x_i) \]

Ejemplo

Definimos la v.a. discreta \[X = \text{"Número de caras obtenidas al lanzar 3 monedas."}\]

  • \[ F(0) = P(X \ge 0) = \frac{1}{8} \]
  • \[ F(1) = P(X \ge 1) = \frac{4}{8} \]
  • \[ F(2) = P(X \ge 2) = \frac{7}{8} \]
  • \[ F(3) = P(X \ge 3) = \frac{8}{8} \]

Propiedades

  • \[ F(x) \in [0, 1] \]
  • Función creciente
  • \[ \lim_{x \to -\infty} F(x) = 0 \]
  • \[ \lim_{x \to +\infty} F(x) = 1 \]
  • \[ P(a \ge X \ge b) = F(b) - F(a) \]
